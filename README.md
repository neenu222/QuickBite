# 🍔 QuickBite

QuickBite is a food ordering and restaurant management web application developed using Python and Django. It allows users to browse food items, place orders, and manage their profiles through an easy-to-use interface.

## 🚀 Features

- User Registration & Login
- Browse Food Menu
- Search Food Items
- Add to Cart
- Place Orders
- Order History
- Admin Dashboard
- Restaurant Management
- Responsive Design

## 🛠️ Technologies Used

- Python
- Django
- HTML5
- CSS3
- JavaScript
- Bootstrap
- SQLite / MySQL

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/neenu222/QuickBite.git
cd QuickBite
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Migrations

```bash
python manage.py migrate
```

### Start Server

```bash
python manage.py runserver
```

Open:

```text
http://127.0.0.1:8000/
```
